Whats intended to be shown on the chart is to illustrate the mean response times for those that accepted and declined their requests, based on gender and personnel type. If I had time references, perhaps I could use some sort of line plot to plot the mean response rates over time.

As for other chart types, maybe some box / violin plots sliced by the conditions mentioned could be an option, especially if I care about skew and response time spreads.

library(ggplot2)
dg_plot <- ggplot(pers_req1, aes(x=gender, y=dec_time, fill = pers_type), ) + 
                  geom_bar(stat = "summary", fun = "mean", position = "dodge2") +
                  theme_bw() +
                  labs(title = "Average Decision Time To Accept or Decline by Gender and Personnel Type") +
                  xlab("Gender") +
                  ylab("Mean Decision Time in Hours")
dg_plot <- dg_plot + facet_wrap(~decision)

dg_plot <- dg_plot + coord_cartesian(ylim = c(0, 6))

return(dg_plot)
